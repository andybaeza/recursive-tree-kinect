# Interactive Recursive Tree with Kinect

An **interactive generative visualization** that grows and transforms in response to human movement.  
Using a **Kinect sensor** and **Processing with Java** with the **SimpleOpenNI** library, this project creates a recursive tree whose **angle, color, and structure** react in real time to the position of the user‚Äôs hands.
